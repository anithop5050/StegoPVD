## 🎉 Release 2.0

### What's New
- **Restructured codebase** with separate `training/`, `production/`, and `utils/` folders
- **Enhanced training pipeline** with multiple training options (optimized and standard)
- **Dataset augmentation** tools for better model performance
- **Production-ready Flask application** in dedicated folder
- **Improved utilities** with modular feature extraction and inference
- **Better gitignore** to exclude heavy model files and datasets

### Features
- ✨ Complete training pipeline with deep learning models
- ✨ Fast and optimized dataset augmentation
- ✨ Production Flask web app for steganography detection
- ✨ SRM filters and feature extraction utilities
- ✨ Comprehensive configuration management

### Installation
```bash
git clone https://github.com/anithop5050/StegoPVD.git
cd StegoPVD
pip install -r requirements.txt
```

### Note
Dataset images are not included in this release. Users should provide their own datasets for training.
